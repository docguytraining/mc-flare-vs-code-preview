# Security Policy

The following versions are supported for reporting vulnerabilities and fixes will be provided. We only support the most recent release with updates. 

| Version | Supported          |
| ------- | ------------------ |
| 0.3.0   | :white_check_mark: |
| <=0.2   | :x:                |
